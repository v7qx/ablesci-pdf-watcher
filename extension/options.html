上传规则
最小自动上传体积 体积低于此值时仅下载和校验。最大值固定为 150 MB。
上传完成后删除本地 PDF 关闭时会保留本地 PDF。
智能推送 提交成功后，按网站返回结果决定是否显示相关文献的智能推送提示。
更多设置
按钮显示名称 显示在 Ablesci 页面按钮上的文字。
按钮颜色 默认状态下的按钮背景和文字颜色。
按钮显示位置 没有快捷应助区域时，回退到 DOI 区域最后。
显示语言 设置插件配置页和应助按钮的显示语言。
保留浏览器下载记录 异常 HTML 下载会保留记录，便于确认。
求助链接当前窗口打开 开启后，求助列表和上传成功后的推荐链接会在当前窗口打开；Ctrl、Command 和中键点击仍保留浏览器默认行为。
ScienceDirect 打开模式 可见模式最接近手动点击；半可见为后台静默 60 秒后切前台；静默模式不会主动切前台。
自动删除 HTML 错误页 默认关闭。开启后，下载到 HTML/登录页/错误页时会尝试删除本地异常文件。
每日应助上限 按 PDF 下载/应助尝试计数，0 表示不限制。用于防止异常时连续下载。
诊断信息 默认关闭。开启后才记录最近一次诊断和出版商链路。

启用 PDF 去水印 在下载 PDF 完成后、上传到科研通之前，自动调用本地去水印工具清理水印。

单文件超时时间 去水印进程的最长允许运行时间（单位：秒）。
去水印出错时的策略 如果去水印失败或超时，是继续上传原 PDF 还是终止上传。

值守参数配置
基础设置
启用低频值守 默认关闭。只在浏览器运行且扩展启用时按低频 alarm 检查。
值守速度模式 控制自动值守的运行速率。极速与快速模式下将获得更密集的应助响应。
月目标 按本月时间进度估算当前应完成量，用于自适应模式下计算调度间隔。
候选筛选
列表 URL 每行一个 Ablesci 求助列表链接。追加 &page_min=1&page_max=5&order=desc 可按页码倒序依次扫描。
非 SD 最低求助量 除 Elsevier / ScienceDirect 外，如果当前出版社列表页统计到的求助量低于这个值，则整页直接跳过，不打开详情页。填 0 关闭，默认 200。
候选筛选 控制哪些求助在列表页和详情页被跳过。
黑名单文件路径 可选。开启黑名单后，留空会读取并自动生成 Helper 本地目录下的 blacklist.txt；填写路径时若 .txt 不存在也会自动创建。读取失败时本次会跳过黑名单检查，不阻断应助。
任务超时 区分未触发下载、下载中超时和任务最长时间；超时会取消当前任务并释放队列。
